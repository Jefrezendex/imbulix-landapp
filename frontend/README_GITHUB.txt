Instruções rápidas para publicar no GitHub Pages

1) Crie um repositório no GitHub com o nome: imbulix-landapp.github.io (recomendado) ou qualquer outro.
2) Faça o push da pasta `frontend/` para a branch `main`.
3) Se usar o repo nomeado `imbulix-landapp.github.io` o site ficará em:
   https://imbulix-landapp.github.io/
4) Alternativamente, crie uma gh-pages branch e ative GitHub Pages nas Settings -> Pages.
5) O frontend faz requests para o backend em:
   https://imbulix-landapp.onrender.com
6) Certifique-se de que o backend esteja ativo (deploy no Render) antes de usar o frontend.
